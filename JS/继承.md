### 原型链继承

```javascript
function Per(){
    this.name = 'Per'
}
Per.prototype = new Person()
```

缺点：原型链继承里面，使用的是同一个内存的值，修改该内存里的值，所有子类的值都会变化。

### 借用构造函数

```javascript
function Per(){
    Person,call(this)
    this.name = 'Per'
}

```
